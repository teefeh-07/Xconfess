## One-Liner Commands

```bash
# Start all services (from repo root)
docker compose -f compose.yaml up -d && npm run dev &

# Quick backend check
curl -s http://localhost:5000/health/live | jq .

# Quick full test suite
./scripts/smoke-test.sh

# Frontend check
curl -s http://localhost:3000/ | head -20
```

## Test Endpoints at a Glance

| Section | Endpoint | Method | Auth | Expected |
|---------|----------|--------|------|----------|
| **Health** | `/` | GET | ❌ | 200 |
| **Health** | `/health/live` | GET | ❌ | 200 |
| **Health** | `/health/ready` | GET | ❌ | 200/503 |
| **Feed** | `/confessions` | GET | ❌ | 200 |
| **Feed** | `/confessions` | POST | ❌ | 201 |
| **Detail** | `/confessions/:id` | GET | ❌ | 200 |
| **Report** | `/reports` | POST | ❌ | 201 |
| **Admin** | `/diagnostics/notifications` | GET | ✅ | 401→ |

**Legend:** ❌ = No auth required | ✅ = Auth required | 401→ = Should fail with 401/403

## Common Issues

| Problem | Solution |
|---------|----------|
| Connection refused | Start backend: `npm run dev --workspace=xconfess-backend` |
| No database data | Create test confession: `POST /confessions` |
| 401 on protected routes | Login first via `/users/login` or `/auth/login` |
| Docker not running | `docker compose up -d` |
| Port already in use | Check `lsof -i :5000` or `:3000` |
